## 2. 注册 NVIDIA NIM 账号 *Content-[00:30]

*   **步骤概览**: 视频提供了详细的注册流程。
*   **详细步骤**:
    1.  打开 NVIDIA NIM 网站（链接在视频下方）。
    2.  点击“Sign In”或“Create Account”。
    3.  填写邮箱地址，点击“Next”。
    4.  设置密码并确认。
    5.  完成人机验证（选择与示例最相似的动物，例如“大象”）。
    6.  点击“Create Account”完成账户创建。
    7.  系统会向注册邮箱发送一个6位验证码（例如：838787），输入验证码后点击“Continue”。
    8.  创建一个用户名（例如：LINGDU）。
    9.  点击“Verify your account to get API access”按钮。
    10. 在弹出的窗口中，选择所在国家/地区（例如：中国），并输入手机号码。
    11. 点击“Send Code via SMS”，系统会发送短信验证码（例如：251981）。
    12. 输入收到的短信验证码，点击“Verify”。
    13. 成功验证后，账户即被激活。

## 3. 获取 API Key *Content-[02:41]

*   **操作流程**:
    1.  登录成功后，点击页面右上角的个人头像。
    2.  在下拉菜单中选择 “API Keys”。
    3.  点击绿色的 “Generate API Key” 按钮。
    4.  在弹出的窗口中，为 API Key 设置一个名称（例如：LINGDU）。
    5.  设置过期时间，建议选择 “Never Expire”。
    6.  点击 “Generate Key”。
    7.  系统会生成一个唯一的 API Key，并仅显示一次，请务必复制并妥善保存。
*   **重要提醒**: 该 API Key 是访问模型服务的凭证，切勿分享或泄露。
*   **截图提示**: *Screenshot-[02:45]* 展示了从个人头像进入 API Keys 页面的过程。*Screenshot-[02:55]* 展示了生成 API Key 的设置窗口。*Screenshot-[03:00]* 展示了最终生成的 API Key 及其复制按钮。

## 5. 配置模型服务 *Content-[04:13]

*   **配置目的**: 将 NVIDIA 提供的模型接入 Cherry Studio。
*   **具体步骤**:
    1.  打开 Cherry Studio，点击顶部菜单栏的“设置”按钮。
    2.  在左侧菜单中找到“模型服务”选项。
    3.  在右侧列表中找到“英伟达 (NVIDIA)”提供商。
    4.  在“API 密钥”输入框中，粘贴之前从 NVIDIA NIM 获取的 API Key。
    5.  点击“+ 添加”按钮。
    6.  在弹出的窗口中，手动添加两个模型：
        *   第一个模型名称：`z-ai/glm4.7`
        *   第二个模型名称：`minimaxai/minimax-m2.1`
    7.  分别点击“添加”按钮将它们加入列表。
    8.  添加完成后，点击“检测”按钮，确保两个模型均显示“连接成功”。
